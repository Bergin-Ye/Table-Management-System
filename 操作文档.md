## 1. 系统概述

金属厂数据管理系统是一套轻量化Web数据管理平台，用于替代传统Excel维护模式，实现数据结构化存储、可视化管理。

### 系统架构

```
浏览器 (Vue 3 + Element Plus)
    ↕ REST API (JWT认证)
后端服务 (Spring Boot 3.3)
    ↕ MyBatis
MySQL 8.4 数据库
```

### 业务模块一览

| 序号 | 模块名称 | 数据量 | 说明 |
|------|----------|--------|------|
| 1 | 送货记录 | 11,580+ | 物料送货全量信息，核心录入表 |
| 2 | 原始记录 | 65,905+ | 金属厂上机/下机原始台账 |
| 3 | 上机物料 | 763+ | 物料上机维修全流程记录 |
| 4 | 送货超比统计 | 186+ | 月度送货超比数据统计 |
| 5 | 结算机台数 | 162+ | 各机型结算机台数统计 |
| 6 | 机型明细 | 3,085+ | 厂房、机台基础信息台账 |
| 7 | 开机数量 | 474+ | 各机型开机数量统计 |
| 8 | 物料表 | 474+ | 物料基础信息（供下拉匹配） |

## 2. 快速入门

### 2.1 访问系统

在浏览器中打开系统地址（由管理员提供），出现登录页面。

### 2.2 注册账号

1. 在登录页点击 **"注册"** 标签
2. 填写：用户名、真实姓名、密码（至少6位）、确认密码
3. 点击 **"注册"** 按钮
4. 注册成功后会切换到登录页

> ⚠️ 新注册的用户默认为**普通用户**角色，需要管理员在"用户管理"中提升为管理员。

### 2.3 登录

1. 输入用户名和密码
2. 点击 **"登录"**

### 2.4 默认管理员账号

| 字段 | 值 |
|------|-----|
| 用户名 | **admin** |
| 密码 | **admin123** |

> ⚠️ 首次登录后请立即修改密码！

### 2.5 界面布局

```
┌─────────────────────────────────────────────┐
│  Header: 系统名称 │ 公司选择 │ 🔔日志 │ 用户头像 │
├────────┬────────────────────────────────────┤
│ 侧边栏  │                                    │
│        │         内容区域                     │
│ 导航菜单 │      (表格/表单/操作)              │
│        │                                    │
└────────┴────────────────────────────────────┘
```

- **顶部栏**: 公司选择下拉框、操作日志快捷入口(管理员)、用户菜单
- **侧边栏**: 各模块导航菜单（可折叠）
- **内容区**: 数据表格、搜索筛选、操作按钮

### 2.6 公司切换

顶部下拉框可以选择公司，切换后所有页面自动加载对应公司的数据。不选择则显示所有公司数据。

## 3. 权限体系

### 3.1 角色说明

| 角色 | 权限范围 |
|------|----------|
| **管理员 (admin)** | 全部页面可见、可操作所有数据、可管理用户 |
| **普通用户 (user)** | 除管理页面外全部可见、仅能修改自己创建的数据 |

### 3.2 权限矩阵

| 功能 | 管理员 | 普通用户 |
|------|:--:|:--:|
| 查看所有业务数据 | ✅ | ✅ |
| 新增数据 | ✅ | ✅ |
| 编辑/删除**自己**创建的数据 | ✅ | ✅ |
| 编辑/删除**他人**创建的数据 | ✅ | ❌ |
| Excel 导入/导出 | ✅ | ✅ |
| 模板下载 | ✅ | ✅ |
| 公司管理 | ✅ | ❌ |
| 操作日志 | ✅ | ❌ |
| 用户管理（角色/密码/删除） | ✅ | ❌ |

### 3.3 数据归属

- 所有新增的数据都会自动记录**创建人**（当前登录用户的真实姓名）
- 每张表格都有"创建人"列，可以识别数据来源
- 修改数据会更新"最后修改人"
- 所有操作都会记录在操作日志中

| 创建人 | 文本 | 自动记录 |

### 5.2 自动计算逻辑

| 字段 | 计算方式 |
|------|----------|
| **年月** | 从日期提取 → `FYyyMM`（如2026年7月 → FY2607） |
| **维修工时** | `结束时间 - 开始时间`，转换为小时 |
| **停机工时** | `结束时间 - 报修时间`，转换为小时 |
| **上次上机时间** | 查询当前下机物料号在本表"上机物料号"列中最后出现的日期 |
| **是否过保** | 上次上机距今 < 6个月 → "未过保"；≥ 6个月 → "已过保"；无记录 → "无" |

### 5.3 筛选条件

额外支持按班次、是否过保筛选。

> ⚠️ 自动计算字段由后端自动处理，前端无需填写，填写了也会被覆盖。

## 15. Excel导入导出

### 15.1 导入模板下载

每个模块页面都有 **"模板"** 按钮，下载含示例数据的Excel模板。

### 15.2 导入数据

1. 点击 **"导入"** 按钮
2. 选择按模板填好的 .xlsx 文件
3. 系统自动校验并导入
4. 导入完成后显示结果：总条数、成功数、失败数
5. 失败的数据会显示行号和失败原因

### 15.3 导出数据

点击 **"导出"** 按钮，按照当前筛选条件导出全部匹配数据为Excel文件。

> 导出会忽略分页限制，导出当前筛选条件下的**全部**数据。

## 16. OCR图片识别

**适用模块**: 送货记录新增页面

### 16.1 使用方法

1. 在送货记录新增对话框中，点击OCR上传按钮
2. 上传包含物料信息的图片
3. 系统自动识别图片中的文本
4. 识别结果自动填充到对应表单字段
5. 人工审核并修正识别误差
6. 确认无误后提交数据

### 16.2 注意事项

- 图片清晰度直接影响识别准确率
- 识别结果需要人工审核确认
- 识别失败或图片模糊时会有明确提示，可重新上传或手动填写

> 注：OCR功能当前处于开发阶段，具体使用方法以后续更新为准。

## 17. 常见问题

### Q1: 忘记密码怎么办？
联系管理员，管理员在"用户管理"中可以重置密码。

### Q2: 为什么我不能编辑某些数据？
普通用户只能编辑/删除**自己创建**的数据。查看"创建人"列确认数据归属。如需修改他人数据，请联系管理员。

### Q3: 为什么侧边栏有些菜单看不到？
"公司管理"、"操作日志"、"用户管理"仅对管理员可见。如果你应该有管理员权限，请联系现有管理员提升你的角色。

### Q4: 导入Excel失败怎么办？
检查以下几点：
- 是否使用了正确的模板格式
- 必填字段是否填写完整（如物料编码不能为空）
- 日期格式是否正确（yyyy-MM-dd）
- 查看导入结果中的失败详情

### Q5: 数据量大时页面慢怎么办？
- 使用筛选条件缩小查询范围
- 调整每页显示条数
- 导出Excel后离线分析

### Q6: 如何查看谁修改了数据？
管理员在"操作日志"中可以查看所有操作记录，包括操作人、时间、变更内容。

### Q7: 自动计算的字段为什么不显示？
维修工时、停机工时、是否过保等字段由后端自动计算。只有在新增或修改数据时才会触发计算，且必须提供必要的源字段（如开始时间、结束时间）。
