## 二、Excel 导入跨天 + 工时（⚠️ 核心验证）

**测试数据** — 在 Excel 中新建一行：

| 日期 | 班次 | 厂房 | 报修时间 | 开始时间 | 结束时间 | 故障现象 |
|------|------|------|----------|----------|----------|----------|
| 2026-07-21 | 白班 | A | 22:00 | 23:00 | 02:00 | 测试故障 |

> 时间列只填 `HH:mm`，不填日期。维修工时和停机工时留空。

**操作**: 原始记录 → 导入 → 选择该 Excel

**查数据库验证**:
```sql
SELECT record_date, repair_request_time, start_time, end_time,
       repair_hours, downtime_hours
FROM original_record ORDER BY id DESC LIMIT 1;
```

**期望结果**:

| 字段 | 期望值 | 失败值 |
|------|--------|--------|
| repair_request_time | `2026-07-21 22:00:00` | 不是 1899 年 |
| start_time | `2026-07-21 23:00:00` | 不是 1899 年 |
| **end_time** | **`2026-07-22 02:00:00`** | `2026-07-21 02:00:00` ❌ |
| **repair_hours** | **`180`**（3小时=180分钟） | `3` ❌ |
| **downtime_hours** | **`240`**（4小时=240分钟） | `4` ❌ |

## 三、超比统计 — 百分比

**操作 1：新增**:
1. 超比统计 → 新增
2. 填入料号: `TEST-PCT-001`（先去156项创建这个料号）
3. 统计日期: `2026-07-21`
4. 单台机用量: `1`
5. **比例(%): `15`**（输入 15，不要带百分号）
6. 含税单价: `100`
7. 类别: `测试`
8. 配件名称: `测试配件`
9. 点击「提交」→ 提示"新增成功"

**操作 2：查数据库**:
```sql
SELECT material_code, ratio FROM delivery_stats
WHERE material_code = 'TEST-PCT-001' ORDER BY id DESC LIMIT 1;
```
期望: `ratio = 0.15`

**操作 3：查列表**:
回到超比统计列表，找到 `TEST-PCT-001` 这条

期望: 比例列显示 **`15%`**（带百分号）

**操作 4：编辑**:
1. 点击该条 → 编辑
2. 查看比例输入框 → 应显示 `15`（不是 0.15）
3. 改为 `20` → 提交
4. 查数据库 → `ratio = 0.20`
